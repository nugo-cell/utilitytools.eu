P2P File Transfer 🔒 DTLS-encrypted · 🚫 no upload · ⚡ direct browser-to-browser

Send any file — photos, videos, PDFs, archives, anything — directly from your browser to another browser. The file never touches our server: it streams peer-to-peer over a WebRTC DataChannel, encrypted end-to-end with DTLS. No size cap, no account, no waiting room.

Start a transfer

Create a private room, share the link with the person you want to send files to, then drop files in. They'll appear on the other side as fast as your connection allows.

🔒 Why this is safer than a normal upload: Sites like WeTransfer or Drive copy your file to their servers. Here, the bytes go straight from your browser to the recipient's browser over an end-to-end DTLS-encrypted WebRTC DataChannel. Our server only relays a tiny "introduction" so the two browsers can find each other — it never sees the file.
