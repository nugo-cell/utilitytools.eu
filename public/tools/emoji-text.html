Emoji Text Generator

Three modes: turn each letter into a regional-indicator emoji, replace keywords with emoji, or sprinkle in random emoji decoration.

Advertisement
Letter mode uses Unicode Regional Indicator Symbols; some letter pairs that match country codes will render as flags. Keyword mode covers ~40 common English words.

FAQ

Why are some letters spelling out country flags?

Two adjacent regional indicators that form a valid country code render as a flag (e.g. US → 🇺🇸).

How does Keyword mode pick emojis?

It runs through ~40 hand-picked English keywords (love, fire, money, cat, pizza, code…) and inserts the matching emoji after each match.
