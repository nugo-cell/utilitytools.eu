Hieroglyphics Name Generator

Phonetic mapping from English letters to Egyptian hieroglyph Unicode. Download your name framed in a cartouche as a PNG.

⚠ Note: This tool is for fun, decorative, and educational use. Hieroglyphs are mapped phonetically — this is not a real translation into Ancient Egyptian.
Each English letter uses one of the standard Egyptian "alphabetic" hieroglyphs (e.g. a → 𓄿, n → 𓈖). The cartouche image is rendered with the Canvas API — no external assets.

FAQ

Is this a real Egyptian translation?

No — phonetic and decorative only. Real Egyptian writing used multi-letter signs, determinatives and grammar this tool does not implement.

What is a cartouche?

An oval frame placed around the name of a pharaoh or important figure to mark and protect it.
