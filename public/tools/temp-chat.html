Create a private chat room, share the link with up to 9 friends, talk in real-time directly between your browsers. Messages flow over WebRTC DataChannels encrypted end-to-end with DTLS — our server only relays a small handshake and never sees your messages. Close the tab and the conversation is gone forever.

Start a chat

Create a room and share the link. Up to 10 people per room. Each pair of browsers builds a direct, encrypted P2P connection — your messages never touch our server.

🔒 What "encrypted" means here: Every WebRTC DataChannel uses mandatory DTLS encryption. Once two browsers in the room have exchanged a tiny "introduction" via our signaling server (SDP + ICE), all chat messages flow directly between them. Even with full server logs, an attacker would only see ICE candidates and SDP offers — never message content.
🔥 Ephemeral by design: No database, no logs, no storage. Messages live only in your open browser tab. Close it = gone forever. Even the room itself is wiped from server RAM the moment the last person leaves.
