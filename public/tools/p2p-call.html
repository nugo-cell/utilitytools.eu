P2P Video Call 🔒 end-to-end encrypted

Free, private 1-to-1 video & audio calls. Create a room, share the link with one other person, talk securely. Audio/video flows directly between your two browsers — never through our server. No account, no app, no recording.

Start a call

Create a brand-new private room and send the link to the person you want to talk to. Up to 2 people per room. Camera and microphone are required — if you only want to talk, use the voice call tool instead.

🔒 What "encrypted" means here: WebRTC mandates DTLS-SRTP encryption for all audio/video between peers. The relay server only forwards a few signaling messages (so the two browsers can find each other) — it never sees, stores, or has the keys for your call. Hosted on DigitalOcean Frankfurt 🇩🇪 · GDPR-friendly · zero logs.

How it works: WebRTC opens a direct, encrypted connection between the two browsers. Our server only relays a small "introduction" (SDP/ICE) so the browsers can find each other through routers/firewalls. We use Google's free public STUN servers. If you're behind a strict corporate or double-NAT mobile network, the connection may fail — try a different network or a personal hotspot.
